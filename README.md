# Simon Say Game
A simple memory game built using HTML, CSS, and JavaScript to practice core JavaScript concepts.

# About the Project
This project was created mainly to strengthen JavaScript fundamentals like:<br>
DOM Manipulation<br>
Event Handling<br>
Functions<br>
Arrays<br>
Game logic<br>

# How It Works
Press any key to start the game. <br>
The system generates a random color sequence.<br>
Repeat the sequence correctly to move to the next level.<br>
The game ends if you click the wrong button.<br>

HTML <br>
